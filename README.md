# 🛞 Tire Shop Mini App

**Консольное мини-приложение для записи в шиномонтаж**  
Без Telegram, без токенов, без интернета. Работает прямо в терминале.

![Demo](screenshot/demo.png)

## 🚀 Возможности

- ✅ Регистрация клиента (имя, телефон)
- ✅ Выбор услуги (4 вида работ)
- ✅ Выбор даты на 7 дней вперёд
- ✅ Выбор свободного времени (9:00–21:00)
- ✅ Подтверждение записи
- ✅ Просмотр своих активных записей
- ✅ Отмена записи
- ✅ Статистика для администратора
- ✅ Тестовые данные уже загружены

## 🛠 Технологии

- Python 3.6+
- Чистый Python (без сторонних библиотек)
- Работает на Windows, macOS, Linux
